## 2. Warning Signs Infographic — How to Spot a Crypto Prop Firm Scam

Vertical list with red warning indicators:

1. "Pays in Own Token" — If you cannot sell it, it is not payment
2. "No Track Record" — Launched in 2025/2026 with no payout history
3. "Unrealistic Promises" — "Make 50% in a week" style claims
4. "No Regulation" — No license, no registered entity, no address
5. "Forums Are Astroturfed" — All positive reviews, no real trader discussions

Each item has a red warning icon. Title: "CRYPTO PROP FIRM — RED FLAGS". Dark background. Amber/red warning theme. 1200x1000px.
